## Chat App

A chat application where the user can send both group and individual messages.

## Project version used

- .NET 8.0 Blazor WebAssembly (WASM)

## Used technologies

- MSSQL
- Identity
- SignalR
- HTML / CSS

## Design Patterns Used

- Data Transfer Object (DTO)
